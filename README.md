# 🍷 Wine Quality Predictor

## 📌 About the Project
This is a Machine Learning web app that predicts whether a wine is of good or bad quality using chemical properties.

## 🚀 Features
- Data visualization 📊  
- Machine learning models 🤖  
- Interactive Streamlit web app 🎯  
- Real-time predictions 🍷  

## ⚙️ Tech Stack
- Python  
- Pandas  
- Scikit-learn  
- Streamlit  
- Matplotlib & Seaborn
- Imblearn

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py

 
## 📂 Dataset
wine.csv is used for training the model.
